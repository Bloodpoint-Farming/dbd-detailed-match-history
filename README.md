## Overview
This userscript makes the [DBD Match History](https://stats.deadbydaylight.com/match-history/) more useful for BP farmers:
- Score Category breakdown (Objectives, Survival, etc.)
- Time between matches (queue, lobby, tally screen)
- BP/hour, timed match-to-match
- Highlights any category < 10K
- Click to Expand/Collapse

## Install
1. Install a userscript manager
2. [Install the userscript](https://github.com/Bloodpoint-Farming/dbd-detailed-match-history/raw/refs/heads/main/dbd-detailed-match-history.user.js)
