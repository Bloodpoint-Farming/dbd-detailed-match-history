# DBD Detailed Match History

## Overview
This userscript adds extra info to the [DBD Match History](https://stats.deadbydaylight.com/match-history/) useful to BP farmers:
- Score Categories (Objectives, Survival, ...)
- Time between matches (queue, lobby, and tally screen)
- BP/hour (match-to-match)
- Highlights any category < 10K
